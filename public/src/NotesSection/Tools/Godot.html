2.2. Modifying Nodes

2.2.1. First Method

We need a node to modify Add a new label Add new text To edit this node we need

  • Reference to the node

To get the reference to the node we just drag and drop the node inside the code editor 

$Label
  • Access the property to change
$Label.text = "Hello"
$Label.modulate = Color.green

Property name is accessed by hovering over it

2.2.2. Second Method

To create a reference to a node inside a variable Hold ctrl while dragging the node to create a var with the reference to the node 

@onready var weapon = $Player/Weapon

onready makes sure that the node is actually created before taking a reference to it $ is a short to getnode() function which takes a relative path To get the absolute path node.getpath()

Path should be avoid because if we change the location of the node or rename it the hardcoded path will throw an error

We can export a node 

@export var my_node : Node

This will give a option in the inspector to choose the specific node

mynode is Node2D to check the type along the inheritance path
